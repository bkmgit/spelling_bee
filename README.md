### Quick Start

1. Clone the repo
  ```
  $ git clone xxxx
  $ cd ggg
  ```

2. Initialize and activate a virtualenv:
  ```
  $ virtualenv --no-site-packages env
  $ source env/bin/activate
  ```

3. Install the dependencies:
  ```
  $ pip install -r requirements.txt
  ```

5. Run the app
  ```
  $ xxxxxxx
  ```
